# 🚀 Arena Top One Checker - Fullstack Application

A Fullstack application designed to retrieve how many times and with which champion a player has already achieved a Top 1 in arena mode on League of Legends.

- 🔗 Frontend: https://github.com/WDFAllan/ArenaTopOneCheckerFrontEnd  

---

## 🧩 Description

This project is a fullstack application composed of:

- a **frontend application** that displays and interacts with this data  

---

## 🛠️ Tech Stack


### Frontend
- React 
- TypeScript
- API consumption (REST)

### General
- JSON (data exchange)
- Git / GitHub

---

## ⚙️ Features

- 🔍 Fetch player data from external sources
- 📊 Analyze player performance
- 🏆 Identify the "Top One" player
- 🌐 Expose data through REST API
- 💻 Display results in a frontend interface
- 🔄 Real-time data interaction between frontend and backend

---

## ▶️ Getting Started

### Prerequisites

- Node.js + npm

---

### 1. Clone the repositories

```bash
git clone https://github.com/WDFAllan/ArenaTopOneCheckerBackEnd.git
git clone https://github.com/WDFAllan/ArenaTopOneCheckerFrontEnd.git
```

### Run frontend

Run the frontend

```bash
cd ArenaTopOneCheckerFrontEnd
npm install
npm start
```
